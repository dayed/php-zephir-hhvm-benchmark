Benchmark for PHP/Zephir/HHVM
=============================

## How to run?

By one command:

```bash
curl -sS https://raw.githubusercontent.com/ovr/php-zephir-hhvm-benchmark/master/benchmark.php | php
```

## Tests

* math
* methodCallEmpty
* methodCallBool
* arrayAppendInt
* arrayAppendString
* readClassConst
* readClassConstBySelf
